Loading an object (.obj) model

An existing object model can be loaded into the AR viewer through an online depository.
An example depository containing a limited range of object files can be found here.

The "Load obj model" tab in the menu defines a marker and a model within a specific directory.
When "import model" is clicked, the defined model is displayed on the defined marker (unless the file does not exist).
When "remove model" is clicked, the defined model is removed from the defined marker.
When "clear marker" is clicked, all models on the defined marker are removed.

The directory from which an object is loaded is defined in Changing sources).
